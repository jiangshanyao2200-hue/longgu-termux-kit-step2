Project Ying (AItermux) Keymap

基本规则：
- 焦点在输入框：按键/触摸优先编辑输入（移动光标、插入、删除等）。
- 焦点在聊天区：按键/触摸优先浏览消息（滚动、选中、展开/收起）。
- PTY 视图打开时：PTY 快捷键优先（PageUp/PageDown/ESC/Home/Alt+Up）。

Chat（通用）
- 输入 `/`：打开命令菜单（↑↓选择，Enter 执行）。
- 命令菜单：`/Cmd` ` /Settings` ` /Help` ` /Switch` ` /SSE` ` /Quit`

Help（/Help）
- ↑/↓：选择条目
- Enter：执行（例如“快捷键”会把本文件注入为系统消息）
- Esc/Home/Left：关闭

输入框（焦点在输入框时）
- ←/→：左右移动光标
- ↑/↓：按“视觉行”上下移动光标（多行输入时）
- 触摸：点击定位光标；拖拽上下左右移动光标
- Enter：发送（粘贴防护期间会转为换行；需要强制发送可用 Ctrl+Enter 或 Alt+Enter）
- Shift+Enter：换行
- Ctrl+Enter：强制入队发送（最低优先级；空闲时自动出队；也可绕过粘贴防护）
- Alt+Enter：强制发送（忙时入队；也用于允许提交占位符粘贴）
- PageUp / PageDown：直接进入“选消息”并切到聊天区（从最后一条开始）
- Tab：全局展开/收起（并切到聊天区）
- Shift+Tab：切换形态（简约 -> 工具 -> 思考 -> 简约）（并切到聊天区）
- End：滚动到底部（并切到聊天区）

聊天区（焦点在聊天区时）
- 触摸：上下滑动滚动；点击某条消息可选中，再点一次取消选中
- ↑/↓：滚动聊天（在“选消息模式”下为选择上一条/下一条消息）
- PageUp / PageDown：进入/推进“选消息模式”
- ←/→：对“选中消息”展开/收起（思考/工具/正文等）
- Tab：全局展开/收起（思考 + 工具详情 + 用户折叠）
- Shift+Tab：切换形态（简约 -> 工具 -> 思考 -> 简约）
- Home：收敛当前选中消息的展开（不改变全局形态）
- Home×2（350ms 内）：全局收敛并退出选中
- End：跳到最新（置底）
- Ins：回到按 End 之前的位置（再次 Ins 不会跳走）
- Alt+↓：复制“选中消息”到剪贴板（termux-clipboard-set）

PTY（内置终端视图，打开后优先处理）
- Home：隐藏 PTY 视图（后台继续运行）
- PageUp / PageDown：在“终端 tab + Chat”环上切换焦点与 tab
  - 焦点切到 Chat 后：↑↓ 用于滚动聊天历史
- Esc：透传给终端内程序（vim/less/cancel 等）
- Esc×2（350ms 内）：结束当前终端任务（kill 当前 PTY tab）
- Alt+↑：把当前终端快照暂存到输入框（由用户决定是否发送给 AI）
- 说明：PTY 默认保持会话；30 分钟自动超时结束；也可用 Esc×2 或 `pty.kill(pid)` 结束

Settings（设置页）
- Esc / Up：返回聊天
- Tabs 焦点：PageUp/PageDown/Tab/←/→ 切换标签；Enter/Down 进入字段
- Fields 焦点：↑/↓ 选字段；Enter 进入编辑；Esc/← 回到 Tabs
- Input 焦点（单行编辑）：Enter 保存；Esc 取消；↑/↓/←/→/Home/End 移动
- Prompt 焦点（多行编辑）：Ctrl+S 保存；Esc 返回；↑/↓/←/→/Home/End 移动；Enter 换行
