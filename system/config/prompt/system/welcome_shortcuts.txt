欢迎使用 Project Ying（萤）。

快捷键（聊天页）：
- ↑/↓：滚动历史（输入中也可）
- Ctrl+↑/Ctrl+↓：输入框多行光标上下
- PgUp/PgDn：输入空→选消息；PTY 开→切终端 tab
- Home：退出选择；若有 Terminal 则切显隐
- ←/→：展开/收起（选中消息时）
- Tab：全部展开/全部收起（输入空时）
- Shift+Tab：详情/简约模式
- Alt+Enter：忙时强发送入队（最低优先）
- Alt+↑：队列（↑↓选 / Enter编辑 / Home退 / 编辑Alt+↓保存）

Terminal（PTY）：
- Home：打开/隐藏 Terminal（后台继续）
- PgUp/PgDn：切换终端 tab
- Esc：发给终端；Esc×2(350ms) 结束任务
- Alt+↑：复制终端快照到输入框
